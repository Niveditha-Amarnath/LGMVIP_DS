Dataset Source : Kaggle

- classification methods:
k Nearest Neighbours
Naive Bayes classifiers
Descision tree classifiers

- There are 4 Features for the classification:
Sepal length
Sepal width
Petal length
Petal width

- There are 3 class for the classification:
Iris setosa
Iris Verginica
Iris Versicolor
